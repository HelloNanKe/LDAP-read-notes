# LDAP简单介绍

LDAP(Lightweight Directory Access Protocol)全程轻量级目录访问协议，是一个访问分布式目录服务的网络协议。主要用于组合和用户管理。

# RFC4510

RFC4510文档是LDAP协议的总纲。总体介绍了LDAP协议技术的路线图，整个协议集的组成，以及与其他规范的关系。
